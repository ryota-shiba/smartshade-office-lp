# SmartShade Office LP

SmartShade（オフィス版）紹介用の1ページLPです。

## ローカル確認

```bash
npm run dev
```

ブラウザで `http://localhost:4173` を開きます。

## Vercel公開

GitHubにこのフォルダをプッシュし、Vercelで新規プロジェクトとしてインポートします。

- Framework Preset: Other
- Build Command: 空欄
- Output Directory: 空欄

フォーム送信先は後日指定のため、現在はフロント側の完了表示のみ実装しています。
